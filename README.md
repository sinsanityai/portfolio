# Sina Werner Portfolio

Portfolio-Website für Webdesign, SEO-Content und AI-assisted Media Projects.

## Stack

- React
- Vite
- React Router
- CSS
- GitHub
- Vercel

## Lokal starten

Wichtig: Die Website funktioniert nicht per Doppelklick auf `index.html`, weil Vite die React-App erst über einen lokalen Entwicklungsserver ausliefert.

```bash
npm install
npm run dev
```

Danach die URL aus dem Terminal öffnen, meistens `http://localhost:5173`.

## Build testen

```bash
npm run build
npm run preview
```
